# UI Kit — JB Digital Consulting Website

High-fidelity recreation of the `jb-digital-site` marketing website.

## Design Context
- Source: `github.com/julijbs/jb-digital-site` (private)
- Language: pt-BR
- Font stack: DM Serif Display (headings) + Inter (body) — aligned with design system
- Background structure: dark navy hero/header → light section → dark section → light → dark footer

## Components
- `Header.jsx` — sticky nav with logo + CTA buttons
- `Hero.jsx` — full-width hero with overline tag + h1 + subhead + CTA pair
- `ServiceCard.jsx` — glass card for service offering
- `PricingCard.jsx` — pricing tier card with features list
- `Footer.jsx` — dark footer with logo + contact

## Screens in index.html
1. Homepage — hero + diferenciais + footer
2. Serviços listing page
3. Service detail — Protocolo Autoridade

## Notes
- Interactive: click nav links to switch screens
- All copy in pt-BR
- Uses Lucide icons (CDN)
